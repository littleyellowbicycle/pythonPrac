提交项目

　　1）或在命令行上创建新的存储库

　　echo "# mydemo" >> README.md

　　git add README.md

　　git commit -m "first commit"

　　git remote add origin git@github.com:sophia1223/mydemo.git

　　git push -u origin master

　　2）或从命令行推送现有存储库

　　git remote add origin git@github.com:sophia1223/mydemo.git

　　git push -u origin master
    
    3)提交修改
    git commit -m "xxxx"
    git push
